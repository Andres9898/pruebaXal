# pruebaXal

## Configuración y Ejecución

1. Clonar el repositorio.
2. Ejecutar el docker compose:

   ```bash
   docker-compose build
   docker-compose up
   ```

3. Entrar a localhost:3000

## Endpoints

localhost:5000
/api/answered-unanswered
/api/highest-reputation
/api/lowest-views
/api/oldest-newest
